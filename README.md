#  Django Library Management System

A modern Library Management System developed with Django.

This project helps users search books, borrow books online, manage accounts, and provides an interactive educational game to improve user experience.

#  Installation

Clone repository

```bash
git clone https://github.com/4tune262/Django-Library-Management-System.git
```

Go to project

```bash
cd Django-Library-Management-System
```

Create virtual environment

```bash
python -m venv .venv
```

Activate virtual environment

Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run migration

```bash
python manage.py migrate
```

Start server

```bash
python manage.py runserver
```

Open browser

```
http://127.0.0.1:8000/
```
